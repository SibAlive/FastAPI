Установка:
1. Клонируйте репозиторий
    git clone <ваш-репозиторий>
    cd secure-todo-api
2. Создайте .env файл в корне проекта:
    SECRET_KEY=ваш_надёжный_секретный_ключ_длиной_32_символа
    ALGORITHM=HS256
3. Соберите Docker контейнер:
    docker build -t my-api .
4. Запустите Docker контейнер
    docker run --rm -p 8000:8000 my-api
5. Перейдите в браузере по ссылке: http://127.0.0.1:8000/docs
6. Получите токен:
    В endpoint'е /login введите логин: admin, пароль: qwerty
    Скопируйте токен, нажмите Autorize и вставьте его в поле Value.


Ответы на вопросы:
1. Docker контейнер запускается в операционной системе Debian 12, так как образ python 3.12-slim основан именное на ней.
2. Docker контейнеры почти всегда используют Linux, так как Docker изначально был разработан для Linux и такие ОС как Windows и MacOS поддерживают Docker только через прослойку (упрощенную версию) Linux.
